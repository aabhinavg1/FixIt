# Basic Terminology in COA MCQs Quiz

<Question question="1). What is a bit?" options={['A group of 8 bytes', 'The smallest unit of digital information', 'A CPU pipeline stage', 'A type of cache']} answer="The smallest unit of digital information" />
<Question question="2). A byte usually contains:" options={['2 bits', '4 bits', '8 bits', '16 bits']} answer="8 bits" />
<Question question="3). In COA, a word usually means:" options={['A human language token', 'The natural data size a processor handles efficiently', 'A cache line only', 'A compiler optimization pass']} answer="The natural data size a processor handles efficiently" />
<Question question="4). What does the CPU execute directly?" options={['High-level source code', 'Machine instructions', 'Comments in assembly', 'Only linker scripts']} answer="Machine instructions" />
<Question question="5). ISA stands for:" options={['Internal System Address', 'Instruction Set Architecture', 'Instruction Scheduling Algorithm', 'Integrated Storage Array']} answer="Instruction Set Architecture" />
<Question question="6). The ISA is best described as:" options={['The software-visible contract of the machine', 'The physical chip package only', 'The operating system kernel', 'The cache replacement policy only']} answer="The software-visible contract of the machine" />
<Question question="7). Registers are important because they are:" options={['The slowest storage in the system', 'Fast storage inside the CPU', 'Always larger than RAM', 'Only used by the compiler']} answer="Fast storage inside the CPU" />
<Question question="8). The program counter stores:" options={['The total number of instructions in the binary', 'The address of the next instruction', 'The current cache size', 'The result of the ALU']} answer="The address of the next instruction" />
<Question question="9). Cache exists mainly because:" options={['RAM is too fast for the CPU', 'The CPU needs frequently used data closer than main memory', 'Registers are slower than RAM', 'Compilers require extra disk space']} answer="The CPU needs frequently used data closer than main memory" />
<Question question="10). A cache miss usually means:" options={['The program has no bugs', 'The needed data was not found in cache', 'The CPU skipped decode', 'The ISA changed during execution']} answer="The needed data was not found in cache" />
